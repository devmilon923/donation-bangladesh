Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the most important concept in web interfaces because, without it, no one can add, read, or update anything.

How do you select an element from the DOM?

Using JavaScript, there are a few ways to select DOM elements. Three of the most commonly used methods are as follows: getElementsByClassName, getElementsByTagName, and getElementById. You will be able to reach any HTML tag, class, or ID element with these selectors. You can use querySelectorAll also- it will enable you to select elements by ID, class, or tag all in one method.

What is event delegation in the context of the DOM, and why is it useful?

When an event occurs on a child element, the event bubbles up to the parent, allowing the parent to handle the event and this is the event delegation in the context of the DOM. This is useful because if you have a list of items, instead of adding a click event to each item. When an item is clicked, you can determine which child was clicked using properties like event.target.

How do you manipulate an element's attributes and styles using DOM?

This is an example how manipulate an element's attributes and styles using DOM:
const element = document.getElementById('myElement');
const value = element.getAttribute('data-custom');
